# OntoFlow - AI数字孪生本体建模平台

## 介绍
基于时序动态的图计算和图谱数据库引擎AbutionGraph，将实体 - 组件 - 系统与基于流的 AI 编程深度结合，构建可演进的数字孪生与世界模型。构建以本体驱动为核心，串联实时数据、业务逻辑与智能 Agent，实现从数据源探知、知识库、记忆编程、智动化工作流到实时业务推理与行动决策的全链路闭环。

![OntoFlow数字孪生全景图](https://i-blog.csdnimg.cn/direct/945b63b63f2943b5803cfdf984ec897b.png#pic_center)
## 核心定位

- 本体驱动：以统一的知识本体为骨架，打通异构数据源与业务系统，让数字世界具备可理解、可推理的语义基础。
- AI 原生架构：原生集成大模型 LLM、RAG、记忆系统与 Pipeline/Agent 能力，让业务系统具备自主感知、推理与行动的智能大脑。
- 全链路闭环：从数据源探知、数据治理、知识库构建、记忆编程，到智动化工作流与实时业务决策，形成完整的智能业务闭环。
- 可演进的世界模型：支持动态扩展实体、关系与规则，让数字孪生体随业务发展持续演进，构建可生长的智能世界。


## 核心能力

- 数字孪生建模：基于实体 - 组件 - 系统范式，快速构建工业、城市、业务等场景的高保真数字孪生体。
- 智能 Agent 编排：通过可视化 / 代码化方式编排智能 Agent，实现复杂业务场景下的自主决策与任务执行。
- 时序图计算：内置高性能时序图计算引擎，支持对动态变化的实体关系网络进行实时分析与推理。
- 记忆与知识管理：提供结构化记忆系统与知识库，支撑大模型进行长上下文、可追溯的智能交互。

## 系统模块和功能介绍

1. 探索       ：全系统功能的智能问答
2. 智能体     ：配置了固定大模型、知识库、数据库、本体库的AgentBot
3. 元数据摄取 ：支持关系型数据库、时序数据库、图谱数据库、NoSQL数据库、键值数据库、向量数据库、数据仓库、大数据平台、搜索引擎、文件数据源、开发框架、API/SaaS服务的连接并自动发现数据资产
4. 元数据资产 ：查看已探查的所有数据源及元数据信息
5. 数据工作流 ：一个AI自动编程实现数据处理的无代码平台
6. 本体建模   ：基于数据流结果自动生成知识图谱，包含实体关系的属性、类型、聚合函数、行动函数
7. 数字孪生   ：基于本体库形成业务感知的行为监控，实际为(实时数据或者定时任务)驱动的图谱查询，并附加到3D可视化场景建模上
8. 知识库     ：GraphRAG的平台实现，不依赖第三方图数据库和向量数据库，实际为基于本体论在OntoFlow上的特别实现
9. 素材库     ：管理系统的可用图片
10. 模型管理  ：接入主流的LLM供应商服务，驱动系统功能
11. MCP     ：接入第三方服务
12. 插件     ：上传自定义的代码文件，如数据类型、聚合函数、行动函数、固定业务的查询函数封装等

备注：OntoFlow 定位为企业级可生产系统，但系统极为精简，只包含3项常规组件：数据库、后端接口程序、前端网页，均为一键部署启动的组件，再无其他需要安装部署的组件。
